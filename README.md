# JavaScript Projects

A collection of beginner-to-intermediate JavaScript projects built while learning core web development concepts such as DOM manipulation, events, timers, API integration, and asynchronous JavaScript.

## Projects

### 1. GitHub Profile Finder (`github_api_project`)

A web application that fetches and displays GitHub user information using the GitHub API.

#### Features

* Search GitHub users by username
* Display profile picture
* Show followers and following count
* Display public repositories
* Fetch real-time data from GitHub API

#### Concepts Used

* Fetch API
* Async JavaScript
* Promises
* JSON Handling
* DOM Manipulation

---

#### Features
