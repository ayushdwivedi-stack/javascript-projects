# JavaScript Projects

A collection of JavaScript projects built while learning and practicing web development.

## Projects

### 1. Color Scheme Switcher
- Change background colors using buttons.
- Concepts: DOM Manipulation, Event Listeners.

### 2. BMI Calculator
- Calculate BMI based on height and weight.
- Concepts: Forms, JavaScript Functions.

### 3. Number Guessing Game
- Guess a random number generated by the computer.
- Concepts: Loops, Conditions, DOM Manipulation.

### 4. Keyboard Checker
- Detect and display key presses.
- Concepts: Keyboard Events.

### 5. GitHub Profile Finder
- Search GitHub users using GitHub API.
- Concepts: Fetch API, Async JavaScript.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)

## Learning Outcomes
- DOM Manipulation
- Event Handling
- API Integration
- Asynchronous JavaScript
- Problem Solving
